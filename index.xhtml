com.io7m.jregions

The jregions package provides a set of immutable area types.

  • Releases
  • Documentation

Features

  • Extensive set of functions for scaling, aligning, moving boxes.
  • 100% automated unit test coverage.
  • OSGi-ready.
  • JPMS-ready
  • ISC license.
